## 📌 Descripción del Proyecto

El objetivo de este proyecto fue desarrollar una página web estática
para la empresa Foca Ice, enfocada en presentar sus productos, mejorar
su presencia digital y ofrecer una interfaz atractiva y funcional para
los usuarios.

---

## 🎨 Diseño e Interfaz

Se implementó un diseño enfocado en:

- Colores relacionados con frescura
  (azules y blancos)
- Interfaz limpia y minimalista - Navegación
  sencilla
- Adaptación a diferentes dispositivos (responsive design)

---

## ⚙️ Funcionalidades Implementadas

- Navegación entre páginas
- Sección de productos
- Enlaces de contacto (teléfono, WhatsApp y correo elecrónico)
- Diseño responsive

---

## 🛠️ Tecnologías Utilizadas

- Live Server

## 📥 Cómo Descargar el Proyecto

### 🔹 Opción 1 – Clonar el Repositorio

Abrir una terminal y ejecutar:

```bash
git clone https://github.com/AlfredoCeron/PaginaWeb-Foca-Ice.git
```

Entrar a la carpeta del proyecto:

```bash
cd PaginaWeb-Foca-Ice
```

---

### 🔹 Opción 2 – Descargar como ZIP

1. Ir al repositorio en GitHub
2. Hacer clic en **Code**
3. Seleccionar **Download ZIP**
4. Extraer el archivo en tu computadora

---

## 💻 Cómo Visualizar la Página en VS Code

### Paso 1 – Abrir el Proyecto

Desde la terminal dentro de la carpeta del proyecto:

```bash
code .
```

Si el comando **_code_** no funciona, abre VS Code manualmente y selecciona
File → Open Folder , luego elige la carpeta del proyecto.

---

### Paso 2 – Instalar Live Server

1. Ir a la sección Extensions en VS Code
2. Buscar “Live Server”
3. Instalar la extensión

---

### Paso 3 – Ejecutar el Proyecto

1. Abrir el archivo `index.html`
2. Clic derecho
3. Seleccionar “Open with Live Server”

El navegador abrirá la página en:

http://127.0.0.1:5500/
